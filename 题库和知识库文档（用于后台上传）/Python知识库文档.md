# Python 知识库文档

本文档用于 AI Interview 项目的 RAG 知识库，内容围绕 Python 后端常见知识点整理，适合上传到知识库进行切分、向量化和语义检索。

---

## 1. Python 基本数据类型

Python 常见的基本数据类型包括 `int`、`float`、`str`、`bool`、`list`、`tuple`、`dict` 和 `set`。  
其中，`list` 和 `dict` 是最常用的可变容器，`tuple` 是不可变序列，`set` 用于去重和集合运算。

## 2. 可变对象与不可变对象

在 Python 中，字符串、数字、元组属于不可变对象，列表、字典、集合属于可变对象。  
可变对象在函数传参或赋值时需要注意引用共享，避免引发意外修改。

## 3. 列表与元组的区别

列表 `list` 可以修改、追加、删除元素，适合存放动态数据。  
元组 `tuple` 创建后不能修改，通常用于表达固定结构的数据，也常用于字典 key。

## 4. 字典的常见用法

字典 `dict` 以键值对方式存储数据，查找效率高，适合表示结构化信息。  
常见操作包括新增、修改、删除、遍历、`get()` 安全读取以及字典推导式。

## 5. 切片操作

切片是 Python 操作序列的一种简洁方式，例如 `a[start:stop:step]`。  
它常用于截取字符串、列表、元组，也可以用于反转序列。

## 6. 函数参数传递

Python 函数支持位置参数、关键字参数、默认参数、可变位置参数 `*args` 和可变关键字参数 `**kwargs`。  
理解参数传递方式，对于设计通用函数和封装工具函数非常重要。

## 7. 匿名函数 lambda

`lambda` 用于定义简单的匿名函数，通常适合短小的一行逻辑。  
它常配合 `sorted()`、`map()`、`filter()` 等函数使用，但复杂逻辑不建议滥用。

## 8. 列表推导式与字典推导式

推导式是 Python 中非常常用的简洁写法，可以快速生成列表、字典或集合。  
例如列表推导式适合过滤和转换数据，字典推导式适合快速构造映射关系。

## 9. 迭代器与生成器

迭代器实现了惰性遍历的能力，生成器则是创建迭代器的一种更简洁方式。  
生成器使用 `yield` 返回数据，适合处理大数据量场景，可以节省内存。

## 10. 装饰器

装饰器本质上是一个函数，用来增强其他函数的行为。  
在 Web 开发中，装饰器常用于登录校验、权限控制、日志记录、缓存等场景。

## 11. 上下文管理器

上下文管理器通过 `with` 语句实现资源的自动申请和释放。  
最常见的场景包括文件打开、数据库连接、锁管理等，可以减少资源泄露风险。

## 12. 异常处理

Python 使用 `try / except / else / finally` 进行异常捕获与处理。  
合理的异常处理可以让程序更稳定，同时提升错误定位能力。

## 13. 面向对象编程

Python 支持面向对象编程，核心概念包括类、对象、属性、方法、封装、继承和多态。  
在项目中，面向对象常用于封装业务实体和服务逻辑。

## 14. 继承与多态

继承可以让子类复用父类代码，多态则体现为同一种接口在不同对象上表现出不同实现。  
在业务开发中，继承可以降低重复代码，多态可以提高扩展性。

## 15. 魔术方法

魔术方法是以双下划线开头和结尾的方法，例如 `__init__`、`__str__`、`__len__`。  
这些方法可以让自定义类具备更符合 Python 习惯的行为。

## 16. 迭代协议

Python 中可迭代对象需要实现迭代协议，常见表现是支持 `for` 循环遍历。  
理解迭代协议，有助于理解生成器、容器类型以及自定义数据结构。

## 17. 模块与包

模块是一个 `.py` 文件，包是包含多个模块的目录结构。  
合理组织模块和包可以让项目结构更清晰，便于维护和协作开发。

## 18. 常用内置函数

Python 提供了很多常用内置函数，如 `len()`、`sum()`、`sorted()`、`min()`、`max()`、`enumerate()`、`zip()`。  
熟悉这些函数可以减少重复代码，提高开发效率。

## 19. 文件读写

Python 通过 `open()` 进行文件读写，常见模式有读、写、追加和二进制读写。  
实际开发中通常结合 `with` 语句使用，确保文件句柄能够被正确关闭。

## 20. Python 异步编程

Python 的异步编程通常基于 `asyncio`，配合 `async` / `await` 使用。  
它适合 I/O 密集型任务，如网络请求、数据库访问和高并发接口处理。
